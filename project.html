Project Description:

The Cloud-Based Waste Management System using Dijkstra’s Algorithm is an intelligent, IoT-enabled solution designed to optimize the collection of municipal waste through efficient routing. The system leverages real-time data, cloud computing, and graph-based shortest path algorithms to enhance the performance and sustainability of urban waste collection.

Objective:

To develop a smart waste management system that: Monitors waste bin status in real time. Uses cloud infrastructure for centralized data management and access. Applies Dijkstra’s Algorithm to determine the shortest and most efficient path for waste collection vehicles. Minimizes fuel consumption, operational costs, and environmental impact.

Reference:research gate

Key Features:

Smart Waste Bins: Equipped with sensors (e.g., ultrasonic, infrared) to monitor fill levels. Transmit data to the cloud in real time.

Cloud Integration:

Centralized platform for storing, processing, and analyzing bin status data.
Provides dashboards and mobile access for waste collection managers.

Route Optimization Using Dijkstra’s Algorithm:

Constructs a graph of bin locations as nodes and roads as weighted edges (distance or time).
Calculates the shortest path for collection vehicles to visit only full or nearly full bins.
Dynamically updates routes based on real-time data.

Web Interfaces:

Interfaces for drivers to view optimal collection routes.
Admin panel for system monitoring and route planning.

Technologies Used:

IoT Devices: Arduino/Raspberry Pi, ultrasonic sensors.
Cloud Platform: AWS / Google Cloud .
Backend: Node.js.
Frontend: React.js / Angular .
Database: MongoDB .
Algorithm: Dijkstra’s Algorithm
